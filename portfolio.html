Trivia Game


Javascript | jQuery | Timers


This is a basic trivia game. It is composed of a trivia form with multiple choice options. In addition, there is a 30 second timer for each question. When the answer is correct a point is tallied and whether the answer is correct or not a gif related to the question displayes afterwards. At the end the score is added up and the user has the option to try again or not.
